# 📚 画廊浏览器 Gallery Viewer

**仿 EhViewer 设计的本地画廊浏览器，专为飞牛 fnOS NAS 打造**

> 沉浸式 Webtoon 滚动阅读 · 智能分类管理 · 动图友好 · 移动端适配
## ✨ 项目简介

**画廊浏览器**是一款运行在飞牛 fnOS NAS 上的本地画廊管理应用，仿照 EhViewer 的核心交互设计，专注于提供**舒适、流畅、专注**的本地图片阅读体验。

不同于在线画廊客户端，**所有图片都来自你 NAS 上的本地文件夹**——零网络依赖、零隐私风险、零合规问题，但保留了与在线阅读器一样顺滑的浏览体验。
### ⚙️ 初始配置

1. 打开应用，点击右上角 **⚙️ 设置**
2. 在「扫描目录」中点击「+ 添加」，填入你存放图片的目录路径，例如：
   - `/vol1/1000/Pictures/Comics` （推荐）
   - `/vol2/1000/Photos/Anime`
3. 点击 **🔄 立即扫描** 按钮
4. 等待扫描完成（首次扫描会生成缩略图，可能需要几分钟）
5. 开始浏览你的画廊吧！

> 💡 **目录结构建议**：每个子文件夹会被识别为一个画廊。如果想分组，把它们放在同一个父目录下即可自动生成「分类」。

## 🎨 功能特性

### 📖 阅读体验（核心）

- **Webtoon 上下滚动模式** 🌟 — 仿 EhViewer 的核心体验，懒加载、连续阅读、完美适配飞牛OS
- **单页 / 双页模式** — 横屏或传统翻页方式
- **沉浸阅读** — 工具栏/进度条 3 秒自动隐藏，按 `H` 键随时切换
- **缩放支持** — 鼠标滚轮、双击、双指捏合
- **键盘快捷键**：
  - `←` / `→` — 翻页
  - `ESC` — 退出阅读
  - `H` — 切换工具栏显隐
  - `F` — 切换全屏
- **进度自动保存** — 关闭后下次打开回到上次位置
- **图片预加载** — 流畅翻页无白屏

### 📚 浏览管理

- **网格视图** — 缩略图瀑布流，纯图片沉浸浏览
- **列表视图** — 含画廊名、标签的紧凑布局
- **分类系统** — 顶层扫描目录下子文件夹自动成为分类
- **FTS5 全文搜索** — 标题 + 描述毫秒级搜索
- **标签云** — 多标签组合筛选
- **高级排序** — 7 种排序方式（最近更新/最新添加/标题/页数/评分等）

### 🏷️ 整理与标记

- **一键收藏** ⭐
- **自定义标签** — 自由创建，颜色可选
- **评分系统** — 1-5 星打分
- **阅读历史** — 最近浏览 10 条
- **批量管理** — 标签管理对话框

### 🎞️ 动图友好

- **封面静态化** — 瀑布流不闪不晃
- **缩略图保留动画** — 多帧 WebP 缩略图
- **内页播放** — 阅读时 GIF/WebP 正常播放
- **智能降级** — 自动检测动画帧数

### 📱 移动端适配

- **响应式布局** — 自适应手机/平板/桌面
- **触摸手势** — 滑动翻页、捏合缩放、双击全屏
- **侧边栏抽屉** — 左滑/点击汉堡菜单打开
- **iPhone 安全区** — 适配刘海屏底部 Home Indicator
- **沉浸式阅读** — 横屏全屏 + 工具栏自动隐藏

### ⚡ 性能

- **缩略图缓存** — 内存 + 磁盘双层缓存
- **懒加载** — 阅读时按需加载图片
- **HTTP 缓存** — 304 Not Modified 减少流量
- **30s 超时保护** — 防止 API 挂死
- **轻量后端** — Flask 单进程支持多用户

## 🐛 常见问题

<details>
<summary><b>Q: 扫描后没有看到画廊？</b></summary>

A: 检查以下几点：
1. 目录是否正确添加（设置 → 扫描目录）
2. 目录权限是否正确（应用以 `gallery_viewer` 用户运行，需授权）
3. 文件格式是否支持（jpg/png/webp/gif/bmp/tiff）
4. 子文件夹中是否真有图片文件

</details>

<details>
<summary><b>Q: 缩略图不显示？</b></summary>

A: 通常是权限问题。fnOS 应用默认以独立用户运行，需要授权目标目录：
- 进入 fnOS 控制台
- 应用中心 → 画廊浏览器 → 权限 → 添加目录

</details>

<details>
<summary><b>Q: 手机上侧边栏打不开？</b></summary>

A: 1.0.2 已修复。点击左上角 **☰ 汉堡菜单** 按钮即可打开。

</details>

<details>
<summary><b>Q: 设置对话框卡死？</b></summary>

A: 1.0.2 已修复。之前是 CSS 规则误伤导致。

</details>

<details>
<summary><b>Q: 动图在画廊卡片里一闪一闪？</b></summary>

A: 1.0.1 起封面/瀑布流固定为静态首帧。点进阅读后动图正常播放。

</details>

<details>
<summary><b>Q: 大目录（> 1000 画廊）扫描慢？</b></summary>

A: 首次扫描会生成缩略图，建议分批扫描。后续增量扫描只处理变更文件，速度快。

</details>
